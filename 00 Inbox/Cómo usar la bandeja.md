# Cómo usar la bandeja de entrada

Esta carpeta es el punto de **captura** del [[Segundo Cerebro]]: cualquier idea, enlace, apunte o pendiente entra aquí sin pensarlo mucho.

## Reglas
1. Capturar rápido, sin organizar.
2. Procesar la bandeja 1–2 veces por semana.
3. Al procesar, cada nota se mueve según el [[Método PARA]]:
   - ¿Tiene fecha límite y resultado concreto? → `01 Proyectos`
   - ¿Es una responsabilidad continua? → `02 Areas`
   - ¿Es útil algún día? → `03 Recursos`
   - ¿Ya no sirve pero quieres conservarla? → `04 Archivo`

Usa la [[Plantilla de Nota Diaria]] para capturas del día a día.
