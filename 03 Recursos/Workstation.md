# Workstation

Setup de trabajo para desarrollo, [[Marca Personal y Contenido|streaming]] y gaming.

## Hardware
- **CPU**: Intel i7 12700K
- **Motherboard**: MSI MAG Tomahawk Z790
- **RAM**: 32 GB DDR4
- **GPU**: NVIDIA RTX 5070
- **Refrigeración**: Corsair H150i iCUE Link 360mm

## Software diario
- **VS Code + Cursor** — driver diario para [[Python]] y web
- **Visual Studio** — desarrollo [[ASP.NET Core]] / C#
- **Postman** — testing y documentación de APIs
- **Azure Portal** — gestión de recursos [[Azure]]
- **PowerShell 7 Preview** — terminal
- **Git + GitHub** — control de versiones
