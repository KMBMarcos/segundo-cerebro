# Astro

Framework web orientado a contenido, usado en el [[Portafolio Web]].

## Por qué lo elegí
- Islands architecture: React solo donde hay interactividad (terminal animada)
- Salida estática ultrarrápida, ideal para un portafolio
- Integración directa con TailwindCSS

## Datos del proyecto
- Versión: Astro 6 con React 19
- Contenido centralizado en un único archivo TypeScript
- Deploy en Netlify / GitHub Pages

## Conexiones
- Aunque soy [[Carrera Backend|backend]], el frontend del portafolio es mi carta de presentación
