# Segundo Cerebro (metodología CODE)

Sistema de gestión de conocimiento personal popularizado por Tiago Forte. Este vault lo implementa junto con el [[Método PARA]].

## El flujo CODE
1. **C**apturar — guardar ideas y hallazgos en la [[Cómo usar la bandeja|bandeja de entrada]] sin fricción.
2. **O**rganizar — mover cada nota a Proyectos, Áreas, Recursos o Archivo según su accionabilidad.
3. **D**estilar — resaltar lo esencial de cada nota; resumir en tus propias palabras.
4. **E**xpresar — convertir el conocimiento en resultados: código, contenido, decisiones.

## Principios
- El conocimiento solo vale si se puede **recuperar** cuando se necesita.
- Las notas se conectan con enlaces `[[...]]` — el grafo emerge solo.
- Preferir notas atómicas: una idea por nota.

## En este vault
- El grafo de conocimiento se genera con **graphify** (`graphify-out/graph.html`).
- La estructura de carpetas sigue el [[Método PARA]].
