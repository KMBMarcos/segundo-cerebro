## Portafolio Web

Sitio personal de una sola página construido con **[[Astro]] 6 + React 19 + TailwindCSS 3**.

## Objetivo
Mostrar perfil, experiencia y proyectos como Backend Developer para conseguir oportunidades laborales y visibilidad como [[Marca Personal y Contenido|creador de contenido]].

## Detalles técnicos
- Todo el contenido vive en `src/data/portfolio.ts` — un solo archivo de datos.
- Fondos decorativos de arte ASCII por sección (`AsciiBackground.astro`).
- Terminal animada de arranque como isla React (`Terminal.tsx`).
- Diseño: fondo `#080808`, acento púrpura eléctrico `#9B30FF`, fuentes Syne y JetBrains Mono.

## Despliegue
- Netlify (`pnpm build` → `dist/`)
- GitHub Pages como alternativa
