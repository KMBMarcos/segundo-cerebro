# Hockjay OS

Distribución Linux personalizada basada en **Linux Mint**, construida con scripts de Shell. Proyecto personal activo (último push: junio 2026).

## Objetivo
Crear una versión de Linux Mint adaptada a mi flujo de trabajo — un sistema operativo propio como proyecto de aprendizaje profundo del ecosistema Linux.

## Dato curioso
Este proyecto tenía una nota vacía (`HockjayOs.md`) en un vault de Obsidian anterior desde diciembre 2025 — la idea llevaba tiempo esperando ser documentada.
